## 테스트 체크리스트

### [BE] API 서버(Swagger) 테스트 체크리스트

#### 회원가입

- [ ] 회원가입 API 호출 성공 확인
- [ ] 필수값 누락 시 에러 응답 반환 확인
- [ ] 중복 username 또는 email 입력 시 예외 처리 확인

#### 로그인 / 인증

- [ ] 로그인 성공 후 JWT 토큰 발급 확인
- [ ] 잘못된 아이디 또는 비밀번호 입력 시 로그인 실패 확인
- [ ] Swagger Authorize에 JWT 토큰 적용 성공 확인
- [ ] 인증이 필요한 API 호출 시 토큰 없이 접근 불가 확인

#### 영화 초기화

- [ ] 영화 초기화 API 호출 성공 확인
- [ ] 초기화 후 영화 데이터가 DB에 정상 저장되는지 확인
- [ ] 초기화 이후 영화 목록 조회가 가능한지 확인

#### 영화 목록 / 조회

- [ ] 영화 목록 조회 API 호출 성공 확인
- [ ] 응답 데이터에 title, release_date, director, genre, poster_url, id 값이 정상 포함되는지 확인
- [ ] 영화가 없는 경우 예외 또는 빈 목록 처리가 정상 동작하는지 확인

#### 리뷰 작성

- [ ] 인증된 사용자로 리뷰 작성 성공 확인
- [ ] 빈 리뷰 또는 잘못된 요청 데이터 입력 시 에러 처리 확인
- [ ] 리뷰 작성 후 해당 영화 리뷰 목록 또는 최근 리뷰 목록에 반영되는지 확인

#### 최근 리뷰 조회

- [ ] 최근 리뷰 조회 API 호출 성공 확인
- [ ] 최근 리뷰 목록 응답이 정상 반환되는지 확인
- [ ] 응답 데이터에 리뷰 내용, 작성일, 작성자 정보가 정상 포함되는지 확인
- [ ] 필요한 경우 영화 제목 정보가 함께 반환되는지 확인

#### 리뷰 삭제

- [ ] 본인이 작성한 리뷰 삭제 성공 확인
- [ ] 다른 사용자가 작성한 리뷰 삭제 시도 시 차단되는지 확인
- [ ] 삭제 후 최근 리뷰 목록 또는 영화별 리뷰 목록에서 즉시 반영되는지 확인
- [ ] 존재하지 않는 리뷰 삭제 요청 시 예외 처리 확인

#### 권한 / 예외 처리

- [ ] 인증 없이 보호된 리뷰 작성/삭제 API 호출 시 차단되는지 확인
- [ ] 잘못된 토큰 또는 만료된 토큰 사용 시 에러 응답 반환 확인
- [ ] 유효하지 않은 path parameter 또는 request body 입력 시 적절한 에러 응답 반환 확인

### [FE] Movies 페이지 테스트 체크리스트

#### 로그인 상태

- [ ] 로그인하지 않은 상태에서 Movies 페이지 진입 시 경고 메시지가 보이는지 확인
- [ ] 로그인한 상태에서 Movies 페이지가 정상 표시되는지 확인
- [ ] 토큰이 만료되거나 잘못된 경우 API 호출 실패가 자연스럽게 처리되는지 확인

#### 영화 목록

- [ ] `/movies/` 호출 후 영화 카드가 4열로 정상 표시되는지 확인
- [ ] 영화 제목이 15자 초과 시 `...` 처리되는지 확인
- [ ] 포스터 이미지 크기가 카드마다 동일하게 보이는지 확인
- [ ] 출시일과 리뷰 작성 버튼이 포스터 아래에 올바르게 붙어 있는지 확인

#### 최근 리뷰 모달

- [ ] `최근 리뷰` 버튼 클릭 시 모달이 열리는지 확인
- [ ] 최근 리뷰가 최대 10개까지 표시되는지 확인
- [ ] 영화 제목, 리뷰 내용, 작성일 표시가 정상인지 확인
- [ ] 모달의 닫기 버튼이 정상 동작하는지 확인

#### 리뷰 작성

- [ ] 영화별 `리뷰 작성` 버튼 클릭 시 해당 영화 기준 모달이 열리는지 확인
- [ ] 빈 내용으로 제출 시 경고가 표시되는지 확인
- [ ] 정상 작성 시 성공 메시지 후 새로고침되어 반영되는지 확인
- [ ] 작성한 리뷰가 최근 리뷰 목록에 나타나는지 확인

#### 삭제 권한

- [ ] 본인이 작성한 리뷰에만 삭제 버튼이 보이는지 확인
- [ ] 다른 사용자가 작성한 리뷰에는 삭제 버튼이 보이지 않는지 확인
- [ ] 삭제 성공 후 목록에서 즉시 사라지는지 확인
- [ ] 삭제 실패 시 에러 메시지가 표시되는지 확인

#### UI 확인

- [ ] 라이트/다크 모드에서 버튼, 텍스트, 포스터 배치가 깨지지 않는지 확인
- [ ] 리뷰 작성 버튼 텍스트가 줄바꿈되지 않고 한 줄로 보이는지 확인
- [ ] 긴 제목, 포스터 없음, 리뷰 없음 같은 예외 데이터에서도 레이아웃이 무너지지 않는지 확인
